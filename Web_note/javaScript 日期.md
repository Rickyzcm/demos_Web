### Date 类型 -- 所属于引用类型

  * 常用语句  

    ```js  
    var now = new Date(); // 调用时无参数则自动获取当前日期时间
    // 有参数调用：若第一个参数是数字，第二个参数就是月份，然后以此类推
    var y2k = new Date(2000,0); // 2000年 1 月 1 日 零时
    var allFives = new Date(2005,4,5,17,55,55); //

    var parseDate = new ( Date.parse("..."));
    // "..."中填的参数形式可以是(以下所填数字均要是阿拉伯数字)
    // 形式1："月/日/年" 
    // 形式2："月(英文名)/日/年" 
    // 形式3："星期(英文名) 月(英文名) 年 时:分:秒 时区(如GMT-0700)" 


    ```
